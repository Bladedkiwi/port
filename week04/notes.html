Date Object

Date is an object with various getter methods for returning information about time like timezone, year, month, etc.

Returns information in local time or Coordinated Universal Time (UTC).

Methods using index start at 1 not 0:

  • getDate(): day of month
  • getUTCDate(): day of month
  • Why use getUTCDate() over getDate? -UTC is a standard that is defined by the scientific community. It's the primary time standard by which the world regulates clocks, much the same as GMT.

Methods using index start at 0:

  • getDay() day of week
  • getUTCDay() day of week
  • getMonth() month
  • getUTCMonth()

Other Methods:

  • getFullYear() - use instead of getYear() because it has broken years after 2000
  • getUTCFullYear()
  • getHours()
  • getUTCHours()
  • getMinutes()
  • getUTCMinutes()
  • getSeconds()/getUTCSeconds()
  • getMiliseconds()/getUTCMilliseconds()
  • getTime()

GetTime() returns a timestamp representing the number in milliseconds.

A day can be represented by 1000 * 60 * 60 * 24 milliseconds

This and That

This points to the object calling a method and allows for generalized methods. Careful with losing scope of this with functions inside functions

That can solve this problem by referencing outside of current scope

For Loops

  • for: loops through a block
  • for/in: loops through properties
  • for/of: loops through values
  • forEach: loops through array and activates function on each element
