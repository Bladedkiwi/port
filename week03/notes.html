Data Attributes

Allows for storing extra information on standard HTML Elements. Can use getAttribute() to read them Simpler way: DOMStringMap reading via a dataset property.

data-key-number = dataset.keyNumber

The dashes are converted to CamelCase in JavaScript.

KeyBoard Events

Event occurs in three ways:

  • Keydown: key pressed, event will continually occur if key is held down
  • Keypress: useful for finding what key was pressed
  • Keyup: occurs when released For older browsers: keycode.js is a normalizer for key codes returned

Query Selectors

  • QuerySelector - finds first element in doc that matches up
  • QuerySelectorAll - finds them all that match

Why they're popular:

  • Called on any element.
  • Customizable ease in precisely selecting items like:
const wonderWoman = document.querySelector('ul ~ p:empty');
IE before ver.8 doesn't support complex css3 notations like in example.

ClassList Property

The ClassList is a List of all classes an element has.

  • add() adds classes to an element without overwriting
  • remove() removes classes specified from element
  • toggle() adds class if it doesn't have that class, and removes it if it does
  • contains() checks if an element has class

For before IE ver.10: Creating a function for adding a class is the workaround. As seen here: 

                
    function addClass(element,newClass) {
        if (element.className) {
            element.className = element.className + ' ' + newClass;
    } else {
        element.className = newClass;
    }
    return element.className;
    }

Window Object

All global JavaScript objects, functions, and variables automatically become members of the window object. ~Naturally.

Surprisingly:
window (the browser viewport) is NOT including toolbars and scrollbars.


    window.open() //opens in a new window
    window.close() //closes current window
    window.moveTo() //move current window
    window.resizeTo() //resize current window
