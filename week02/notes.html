    Common/Best Practices Ch. 4

    • Declare Function literals, function declarations or function expressions
    • Don't repeat yourself - every part of a program only written once
    • rest operator - Collects all the arguments entered into an array. function rest(...args) { return args; }
    • Common to use a for of loop to iterate over each of the arguments entered.
    • Useful for an improved mean function: function mean(...values) { let total = 0; for (const value of values) { total += value; } return total/values.length; }
    • Default Parameters- Assigning values on default function hello(name= 'World') { console.log(`Hello ${name}!`); }
    • -Golden rule: Always put default parameters after all the other parameters Arrow Functions const square = x => x*x;
    • -less verbose -single parameters don't need parenthesis
    • -body of function doesn't need placing inside a block
    • -return keyword not required
    • -don't bind their own value of this to the function
