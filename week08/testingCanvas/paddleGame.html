Pong

The journey

I delved deeper into what purpose canvas has, and I believe I have found it. Below, is a game of Pong I translated from a Python project I did for cs241. I was surprised to learn that the cicles, paddles, and x,y coordinates are defined in a similar way between Python and JavaScript. The overall translation went rather smoothly except some hiccups that occured with the paddle movement, and working out how to update the board. This game is also very low on rescources coming in at only 70kb. Granted it is very simple, but it does mean making a game is fairly simple for the browser.
